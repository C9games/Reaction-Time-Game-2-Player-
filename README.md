# Reaction-Time-Game-2-Player-
A 2 player game to test who can react faster.

I know there are other reaction time games pretty much identical to this one out there but I just decided to make my own without following tutorials :).

(schematic correction: pin 8 does not go to the buzzer, it goes to the red xled, and pin 2 goes to the buzzer, not the xled. In the photo the pins are joined together.)
